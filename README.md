[Кольоровий зір](https://github.com/3demax/ColorVision)
==============

Программа "Кольоровий зір" разроблена спеціально для офтальмологічного відділення Дніпропетровської міської лікарні в Україні.


Інструкціі
----------

Для початку тесту оберіть "Режим пацієнта". На єкрані з’явиться тестове зображення, опис завдання, яке потрібно виконати, та поле для вводу відповіді.
У будь-який час під час тесту лікар може перейти до "Режиму лікаря", щоб подивитися результати або відредагувати відповіді.

Демо
----

Спробувати программу можна за адресою at http://3demax.github.com/ColorVision
Оригінальні зображення тимчасово видалені. Дозвіл на використання уточнюється.


[Color Vision](https://github.com/3demax/ColorVision)
==============

Description
-----------

Сolor vision is a program for color-blindness detaction developed for Dnipropertovsk Hospital and in famous in ex-USSR "Rabkin tables" color blindness test.
